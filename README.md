# CCNA Lab Portfolio

This repository contains hands-on labs completed while studying for the CCNA certification. Each lab demonstrates practical networking skills, troubleshooting, and an understanding of fundamental concepts required for a NOC Technician role.

---

## Labs

### Lab-01-Basic-Network

* Built a simple LAN using a switch and verified connectivity between two PCs.
* Focused on IP addressing, basic connectivity, and using `ping` as a troubleshooting tool.

### Lab-01.5-Mini-ARP

* Explored ARP (Address Resolution Protocol) to understand how devices map IP addresses to MAC addresses.
* Demonstrated observation of ARP tables and packet flow in simulation mode.

### Lab-02-OSI-Model-&-Packet-Flow

* Investigated the OSI model and observed how data travels across layers.
* Used simulation mode to watch packet encapsulation, ICMP requests/replies, and protocol interactions.

### Lab-03-Subnetting-&-Inter-Network-Communication

* Configured two networks connected via a router.
* Verified connectivity, learned default gateways, and performed step-by-step troubleshooting including `show ip interface brief`.

## Goal

To build practical networking skills, understand how networks operate at multiple layers, and demonstrate readiness for entry-level networking roles such as NOC Technician.
