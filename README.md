# CCNA Lab Portfolio

This repository contains hands-on labs completed while studying for the CCNA certification.

## Labs
- Lab 01 – Basic Network Connectivity  
- Lab 01.5 – ARP  
- Lab 02 – OSI Model & Packet Flow  
- Lab 03 – Subnetting & Routing  

## Tools Used
- Cisco Packet Tracer  

## Goal
To build practical networking skills and demonstrate readiness for a NOC Technician role.
